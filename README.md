# 🦇  PORTFOLIO
### [ AISHWARYA RANGU ]

A high-fidelity, interactive professional portfolio inspired by the tactical HUD of the Batman universe. 

## 🌐 LIVE_DEPLOYMENT
**Official Link:** [https://aishwarya-rangu-portfolio.vercel.app/](https://aishwarya-rangu-portfolio.vercel.app/)

---

## 🏎️ KEY_FEATURES
- **Gotham Pursuit**: Interactive Batmobile project navigation.
- **The Codex**: Tactical manuscript deck for professional writings.
- **22-Node HUD**: High-speed tech stack visualization.
- **Mission Terminal**: Real-time social connectivity and secret protocols.

## 🛠️ TECH_STACK
- **Frontend**: Next.js 14, Tailwind CSS
- **Motion**: Framer Motion
- **Sound**: Hans Zimmer (Dark Knight Theme)

---

## 🚦 QUICK_START
```bash
npm install
npm run dev
```
